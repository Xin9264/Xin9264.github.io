静态变量 Static Typing

java的一个最重要的特征就是语句需要static type.Java变量可以包含type的值，同时也可以只包含type。一个变量的type是永远不可以改变的

static type 有以下优点：

  • 便于程序员debug
  • 不用担心type error
  • 每个变量、参数和方程都有一个明确的类型，因此程序员可以很轻松理解代码

Class Methods vs. Instance Methods

类方法与实例方法

java允许我们定义两个类型的方法

类方法，也就是静态方法

实例方法，也就是非静态方法

实例方法只能往类中传入一个特殊的实例。而静态方法是类本身采取的行动。这两种方法在不同情况下都是有用的。
